# K230 手势识别扩展板 PCB 设计

## 功能

插在庐山派 40P 排母上的扩展板，手势控制两路继电器通断。

## 手势联动

| 手势 | 继电器1 | 继电器2 | 红灯 | 绿灯 |
|------|--------|--------|------|------|
| fist ✊ | 吸合 | 断开 | 亮 | 灭 |
| five 🖐 | 断开 | 吸合 | 灭 | 亮 |
| yeah ✌️ | 断开 | 断开 | 灭 | 灭 |
| 其他 | 保持 | 保持 | 保持 | 保持 |

## GPIO 分配

| 引脚 | 功能 | 方向 |
|------|------|------|
| GPIO42 | 继电器1（经 S8050 驱动） | OUT |
| GPIO46 | 继电器2（经 S8050 驱动） | OUT |
| GPIO47 | LED 红（继电器1状态） | OUT |
| GPIO21 | LED 绿（继电器2状态） | OUT |
| GPIO17 | 按键（PULL_UP） | IN |
| Pin2/4 | 5V 供电 | — |
| Pin6/9 等 | GND | — |

## 电路设计

### 每路继电器驱动

```
GPIO ──[1KΩ]──→ S8050 基极(B)
                  │
5V ──[继电器线圈]──→ 集电极(C)
  │                  │
  └──[1N4007 ←]─┘  发射极(E) ──→ GND
     续流二极管
```

### 每路状态灯

```
GPIO ──→ LED ──[220Ω]──→ GND
```

### 按键

```
GPIO17 ──→ 按键 ──→ GND
（MCU 内部 PULL_UP，按下=低电平）
```

### 去耦电容

```
5V ──[100nF]──→ GND
```
